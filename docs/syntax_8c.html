◆ syntax_convert_to_color()

int32_t syntax_convert_to_color ( int  highlightGroup)

Get the rgb color value of a syntax highlighting group.

Parameters
highlightGroupThe highlight group where the corresponding color value is determined
Returns
A 32 bit value where the 3 least significant bytes control the displayed color and the most significant bit is used to detetemnine whether the highlighting applies to the fore- or the background.
